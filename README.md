# Sun8min

Sun8min的个人项目，在线实例 [sun8min.com]，详见博客 [Blog]

---
## 功能简介

- 跨域名单点登入登出
- 分布式事务
- 支付宝PC/Wap端支付

---
## 启动项目

- 启动 注册中心：[zookeeper-3.4.13]
- 启动 分布式事务支持：[fescar-server-0.4.0]
- 启动 nosql：[redis-5.0.4]
- 启动 cas 服务端: [cas-server-5.3.9]
- 启动 pay, account, order, product, user 微服务模块
- 启动 seckill 微服务模块
